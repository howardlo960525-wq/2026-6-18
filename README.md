# PlayMate 遊戲教練平台

PlayMate 是一個前端互動原型，提供教練媒合、聊天室、預約、影片分析與訂閱方案切換。

## 功能

- 20 位教練資料，依現有教練設計延伸出不同屬性、遊戲、評分與價格。
- 每個功能選單都有自己的區塊：總覽、教練媒合、聊天室、預約、影片分析、訂閱。
- 聊天室可選擇教練、使用快速訊息，並將聊天紀錄存在 `localStorage`。
- 教練媒合可依屬性、遊戲、搜尋關鍵字與需求標籤重新排序。
- 影片分析目前為本地規則整理，不需要後端 API 或 API key。

## 啟動

如果電腦有 Python：

```powershell
python main.py
```

開啟：

```text
http://127.0.0.1:8000
```

也可以直接用瀏覽器開啟 `index.html`，聊天室與互動功能都會在本機執行。
